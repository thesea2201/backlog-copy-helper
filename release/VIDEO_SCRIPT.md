# Demo Video Script for Chrome Web Store

**Target length:** 30-60 seconds  
**Resolution:** 1280x800 (screen recording, 16:10 aspect ratio)  
**Format:** MP4, max 20MB  
**Voice:** Optional text overlays + background music, or casual voice-over.

---

## Scene 1: Hook (0:00 - 0:05)

**Visual:** Fast zoom into a Backlog issue page. Cursor hovers over the cyan "Copy MD" button.  
**Text overlay:** "Backlog is great. But copying issue info is tedious."  
**Action:** Click "Copy MD". Cut to a Slack message input where the markdown is pasted instantly.

## Recording Tips

1. **Use a clean Backlog demo project** with realistic issue titles and comments.
2. **Hide personal data**: blur any real names, emails, or internal project names.
3. **Highlight cursor**: use a screen recorder that adds a visible click ripple (e.g., OBS + click effect, or Loom built-in).
4. **Smooth transitions**: use quick cuts (0.5s fade) between scenes to keep energy high.
5. **Text overlays**: Use bold, high-contrast sans-serif font (e.g., Inter, Roboto) at 32-48pt. White text with a subtle black shadow for readability.
6. **Music**: Upbeat, royalty-free lo-fi or ambient tech track. Keep volume low if using voiceover.
7. **No audio?** Fine — text overlays + energetic pacing work great for store videos.

---

## Recommended Tools

- **Screen recording:** OBS Studio (free), Loom, or Screen Studio (Mac)
- **Video editing:** DaVinci Resolve (free), iMovie, or CapCut
- **Text overlays:** Built into most editors, or use Canva to design static title cards
- **Compression:** HandBrake (free) to compress to <20MB while keeping quality
