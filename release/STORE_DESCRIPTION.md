## Short Description (up to 132 characters)
Copy issues as markdown & translate Backlog content with one click. Supports Google Translate + Gemini.

## Full Description
Backlog Utils supercharges your [Backlog](https://backlog.jp/) workflow with one-click utilities designed for developers, project managers, and multilingual teams.

**Copy Issue as Markdown**
On any Backlog issue page, a "Copy MD" button appears next to the issue key. One click copies the issue key, title, and summary as clean markdown — perfect for pasting into Slack, GitHub PRs, Notion, or documentation.

**Copy Notifications to Spreadsheet**
Click the "Copy Noty" button next to your notifications bell to instantly copy this month's notifications as tab-separated values. Import directly into Excel, Google Sheets, or any tracker to build status reports in seconds.

**Translate Anything on Backlog**
Every comment and article gets a "Translate" button. Choose from 10 languages (English, Japanese, Vietnamese, Spanish, French, German, Chinese, Korean, Russian, Portuguese). The extension uses Google Translate by default and can optionally route through Gemini for context-aware, technical translations.

**Smart Settings**
- Pick your default target language from the popup.
- Control translation caching: Never cache, 7 days, 1 month, or forever.
- Write a custom translation prompt (e.g., "Keep programming terms in English").
- Reset your Gemini session anytime.

**Privacy First**
- No sign-up required.
- No analytics or tracking.
- All settings and translation cache stay in your browser.
- Translation text is sent directly from your browser to Google Translate / Gemini; we never see it.

Backlog Utils is open source. If you find a bug or have a feature idea, please open an issue on our repository.
