# Screenshot & Asset Guide for Chrome Web Store

## Required Assets

| Asset | Size | Quantity | Notes |
|-------|------|----------|-------|
| Screenshot | 1280x800 or 640x400 (16:10) | 3-5 | PNG or JPEG, no transparency |
| Promotional Tile - Small | 440x280 | 1 | Optional but recommended |
| Promotional Tile - Large | 920x680 | 1 | Optional but recommended |
| Promotional Tile - Marquee | 1400x560 | 1 | Optional |
| Icon | 128x128 | 1 | Already have `icons/icon128.svg` |

## Recommended Screenshot List

### Screenshot 1: Copy MD Button on Issue Page
**Scene:** A Backlog issue page with the cyan "Copy MD" button visible next to the issue key.
**Caption:** One-click copy issue key, title & summary as markdown.
**Tip:** Use a real or demo issue with a meaningful title so the value is obvious.

### Screenshot 2: Markdown Pasted Elsewhere
**Scene:** A Slack message, GitHub comment, or Notion page showing the pasted markdown output.
**Caption:** Paste clean markdown into Slack, GitHub, Notion, or docs.
**Tip:** This demonstrates the *result* of the feature, not just the button.

### Screenshot 3: Copy Notifications
**Scene:** Backlog header with the "Copy Noty" button next to the bell icon, plus a spreadsheet showing the imported TSV data.
**Caption:** Export monthly notifications as spreadsheet-ready data in one click.

### Screenshot 4: Translate in Action
**Scene:** A Backlog comment with the "Translate" button and the translated block shown below the original text.
**Caption:** Translate comments & articles into 10 languages instantly.
**Tip:** Pick a Japanese comment translating to English so the before/after is visually clear.

### Screenshot 5: Popup Settings
**Scene:** The extension popup showing the Main tab (language selector, cache settings) and the Settings tab (custom prompt, reset Gemini).
**Caption:** Customize target language, cache duration, and translation prompts.

## Video (Optional but Recommended)
- **Format:** MP4 or WebM
- **Max size:** 20MB
- **Suggested length:** 30-60 seconds
- **Content:** See VIDEO_SCRIPT.md for a ready-to-use script.

## Tips for High Conversion
1. Use real data in screenshots (not Lorem Ipsum).
2. Show the *outcome* of using the extension, not just the UI.
3. Keep screenshots at exactly 1280x800 for crisp display on the store.
4. Add short, benefit-driven captions directly in the image if desired (Chrome Web Store also supports text captions).
